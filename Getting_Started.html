Django Chat Bot

Welcome to the Django Chat Bot project! This web chat AI bot utilizes the OpenAI API to provide interactive and intelligent responses. Follow the steps below to set up and run the project on your local machine.

Getting Started

To get started with this project, ensure you have the following installed:

  • Python 3.x
  • pip
  • Django
  • Any additional dependencies listed in requirements.txt

Installation

  1. Clone the Repository: 
    git clone https://github.com/Aijcoder/DjangoChatBot.git
cd DjangoChatBot
  2. Create a Virtual Environment: 
    python -m venv venv
source venv/bin/activate  # On Windows use `venv\\Scripts\\activate`
  3. Install Dependencies: 
    pip install -r requirements.txt

Setup

  1. Configure Environment Variables:
    • Create a .env file in the project root directory.
    • Add your environment-specific settings and secrets (e.g., OpenAI API keys) to the .env file. Ensure that this file is not included in version control for security reasons.
  2. Collect Static Files: 
    python manage.py collectstatic

    Follow the prompts to confirm the collection.

  3. Apply Migrations: 
    python manage.py migrate

Running the Project

  1. Start the Development Server: 
    python manage.py runserver
  2. Access the Application:

    Open your web browser and navigate to http://127.0.0.1:8000 to see your application in action.
